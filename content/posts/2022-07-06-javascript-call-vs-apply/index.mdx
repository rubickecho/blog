# call()

## call() 是什么?

调用一个传入给定this和参数的方法

> **注意：**call()方法的作用和 apply() 方法类似，区别就是`call()`方法接受的是**参数列表**，而`apply()`方法接受的是**一个参数数组**。
> 

```jsx
fn.call(thisArg, arg1, arg2, argN)
```

# apply()

## apply() 是什么？

调用一个传入给定this和参数的方法，和call作用一样，不同的是传入参数为数组或类数组对象

```jsx
// 语法
fn.apply(thisArg, [argsArray])
```
